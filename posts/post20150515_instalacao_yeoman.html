Para que eu não fique mais sofrendo na hora de instalar o YEOMAN em uma nova máquina para rodar um projeto já existente segue os somandos a serem execultados no Ubuntu: 

# sudo apt-get install nodejs nodejs-legancy npm git
# sudo npm install -g yo
# sudo npm install -g grunt
# sudo npm install -g grunt-cli
# sudo npm install -g bower

Então na pasta do projeto deve-se dar os seguintes comandos: 

$ npm install
$ bower install
$ grunt serve

Caso de erro de Maximum call stack size exceeded digite o seguinte comando: 

$ echo fs.inotify.max_user_watches=524288 | sudo tee -a /etc/sysctl.conf && sudo sysctl -p

Caso de erro ao executar o bower install não terminando de baixar as dependências, execute o seguinte comando. 

$ git config --global url."https://".insteadOf "git://"
